### Run Locally

To run **FastLane** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/alphaotuken/fastlane.git
```

Windows:

```bash
git clone https://github.com/alphaotuken/fastlane.git
```
